==Release notes_Intel Clear WArabic==

Date of creation: 2014-05-19

Changes since previous release:
- Font name changed to Intel Clear WArabic


Basic information
1 Basic family name: Intel Clear WArabic
2 TrueType Version record: 1.101
3 Regional package: Arabic
4 Supported scripts: Latin, Arabic
5 Number of glyphs: 1345
6 Weights/Styles: Light, Regular, Bold
7 Usage: Text and Display
8 Non-Latin characters sets, except Chinese, include a basic set of Latin characters as standard.  Korean, Thai Traditional, Thai Modern, Devanagari, Hebrew, Arabic and Japanese are missing the following glyphs shown below; they can be accessed by downloading the Latin font set.

Technology
1 Files formats: The files have been delivered in the following formats: TrueType-based OpenType; EOT; WOFF.
2 Embedding settings: Everything is allowed (installable mode); subsetting is allowed.
3 Hinting information: All glyphs are manually hinted for optimized rasterization in Microsoft ClearType or for other LCD devices.
